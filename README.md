The Hymn of Death | Cinematic Tribute Website

This project is a front-end web development showcase designed as an immersive tribute to the 2018 K-drama The Hymn of Death (based on the true story of Yun Sim-deok and Kim U-jin).

Key Features:

Immersive Design: A 1920s-inspired "Noir & Gold" aesthetic with film grain overlays and smooth scroll animations.

Cinema Mode: A toggleable feature that dims the interface for a focused viewing experience.

Media Center: A custom tabbed video player for trailers, episodes, and fan edits.

Audio Visualizer: A functional music player for the OST with CSS animation.

Interactive Gallery: A masonry-style gallery with a lightbox view.

Tech Stack:

HTML5

CSS3 (Custom Animations + Tailwind CSS)

JavaScript (Vanilla)
